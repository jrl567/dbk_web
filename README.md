## dbk_web

This repository contains the source code for the official website of **The Dandy Buzzkills**.

The site is intentionally built using raw HTML, CSS, and minimal JavaScript, inspired by the aesthetic and simplicity of late-90s and early-2000s web design.

Because the site is fully static, it can be hosted on any basic web server or static hosting service.

Technologies used:

- HTML
- CSS
- Minimal JavaScript

No build system, frameworks, or external dependencies are required.
